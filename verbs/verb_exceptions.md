# Verb exceptions list 

## Congugare 1

### The "short ones"

#### a da

| _**Pronoun**_ | _**Verb**_ | _**Comment**_ |
| --- | --- | --- |
| Infinitive | a da | "To give" |
| Eu | dau | - |
| Tu | dai | - |
| El / Ea | dă | - |
| Noi | dăm | - |
| Voi | dați | - |
| Ei / Ele | dau | Same as **Eu** *(not as El / Ea!)* |
